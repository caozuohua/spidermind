# Spidermind Examples

This directory contains example scripts demonstrating how to use Spidermind.

## Available Examples

- `basic_crawler.py` - Basic web crawler example
- `e_commerce_scraper.py` - E-commerce site scraper

## Running Examples

```bash
python examples/basic_crawler.py
```

## Contributing Examples

We welcome contributions of new examples! Please:

1. Place your example in this directory
2. Include a docstring explaining what the example does
3. Add comments explaining key concepts
4. Test the example before submitting
